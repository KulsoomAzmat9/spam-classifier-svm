# Spam Classifier - Support Vector Machine

This is a Spam Email Classifier built using *SVM Algorithm*.

## Accuracy
*84.62%*

## Features
- Clean and preprocess email data
- Train SVM model
- Predict if an email is Spam or Ham
- Deployed with Streamlit

## How to Run
1. Install requirements: pip install -r requirements.txt
2. Run app: streamlit run app_svm.py

## Dataset
email_data.csv
